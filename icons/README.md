# Icons Directory
# 图标目录

This directory should contain PWA icons for the app.
此目录应包含应用的PWA图标。

## Required Icons
## 所需图标

The following icon sizes are required for the PWA manifest:
PWA manifest需要以下尺寸的图标：

- `icon-72.png` (72x72)
- `icon-96.png` (96x96)
- `icon-128.png` (128x128)
- `icon-144.png` (144x144)
- `icon-152.png` (152x152)
- `icon-192.png` (192x192)
- `icon-384.png` (384x384)
- `icon-512.png` (512x512)

## Creating Icons
## 创建图标

### Option 1: Use Online Tools
### 选项1：使用在线工具

1. Create a 512x512 icon image (tennis ball, racket, or your design)
2. Use [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator) or similar tool
3. Generate all required sizes

### Option 2: Manual Creation
### 选项2：手动创建

1. Design a 512x512 icon
2. Resize to all required sizes
3. Save as PNG files with correct names

### Option 3: Use Placeholder
### 选项3：使用占位符

For testing, you can use a simple colored square or download free tennis icons from:
测试时，可以使用简单的彩色方块或从以下网站下载免费的网球图标：

- [Flaticon](https://www.flaticon.com)
- [Icons8](https://icons8.com)
- [Font Awesome](https://fontawesome.com)

## Apple Touch Icon
## Apple触摸图标

The `icon-192.png` is also used as the Apple touch icon for iOS home screen.
`icon-192.png`也用作iOS主屏幕的Apple触摸图标。

## Note
## 注意

If icons are missing, the PWA will still work but may not display properly when installed.
如果缺少图标，PWA仍可工作，但安装后可能无法正确显示。
